To attach a file from another app (e.g. Mail, Dropbox) to an exisiting expense item or a brand new expense item, follow the instructions below:

Step 1*

Open the file from the other app and tap on the 'share' button (usually at the top right corner). Once the 'Share' menu pops up on the screen, tap on 'Open in EZBooks'.

Step 2

EZBooks will be opened with 2 options:
1. Create a new expense item with the incoming file.
2. Attach the incoming file to an existing expense item.

Step 3

Choose 'Attach to existing' if you'd like to attach the file to an existing expense item, then tap on the expense item. The file will be attached to the selected expense item.

Choose 'Create new' to create a brand new expense item with the file.

*Note

You may not see the 'share' button in some of the apps, for example, in Google Drive app, once you open the file, you will see a similar 'open in' button.
